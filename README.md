# Running the AP
``` go run main.go ```

## Access the API endpoints:
To count runes:
``` http://localhost:8080/count?text=Hello&text=World ```
 
## To find unique strings:
``` http://localhost:8080/unique?text=Hello&text=World&text=Hello ```
